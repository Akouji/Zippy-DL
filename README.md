# Zippy DL

Node.js Zippyshare Downloader CLI

### Installing

Clone This Repository

```
$ git clone https://github.com/anasrar/Zippy-DL.git
```

Change Directory

```
$ cd Zippy-DL
```

### Install Dependencies

```
$ npm install
```

### How To Use

Get List Anime From Homepage
```
$ npm app.js <url>
```
