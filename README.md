# Zippy DL

Node.js Zippyshare Downloader CLI

# Install Via NPM

```
$ npm install zippydl -g
```

# Install Via Github

Clone This Repository

```
$ git clone https://github.com/anasrar/Zippy-DL.git
```

Change Directory

```
$ cd Zippy-DL
```

### Install Dependencies

```
$ npm install
```

# How To Use

Basic Use (Install Via Github)
```
$ npm app.js <url>
```
Install Global
```
$ zippydl <url>
```
